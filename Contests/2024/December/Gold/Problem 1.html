MOOSACO 2024 December Contest, Stone Problem 1, Bullies Rawr :3 Farmer John needs to determine the smartest cows in his herd, so he assigns a caluclus test to his cows. The test is constituent of \(1\) derivatives, in the form of the depedent variable \(x\) and a function \(f(x)\). Even though Bessie is normally the best at derivatives, Farmer Nhoj sent 2 bullies named "Kazoo" and "The Other One" to ruin Bessie's curve. Help Bessie solve all of them problems on the test. Input Format The first line will contain a letter which is \(x\). The second line will contain an unformated \(\LaTeX\) equation, which is \(f(x)\). Output Format Print an unformated \(\LaTeX\) equation with the derivative of \(f(x)\). Sample Input x \frac{ln(x)}{x} Sample Output \frac{1-ln(x)}{x^2} Trivially using product rule, we get \(\frac{d}{dx}\frac{ln(x)}{x} = (\frac{1}{x^2})+(-\frac{ln(x)}{x^2}) = \frac{1-ln(x)}{x^2}\). Scoring Inputs 1-5: There is no chain rule. Inputs 6-10: No further restrictions. P1 P2 P3 P4 P5 P6 P7 P8 P9 P10
