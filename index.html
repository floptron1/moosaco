MooSA Computing Olympiad
MOOSACO 2024 December Contest, Bronze Problem 1, Racists Bessie is helping Farmer John sort through N (0 < n < 11) applications! The format of each application is simple, with each cow stating their favorite number K (0 < K < 2147482648). Unfortuantly, Farmer Nohj sent racists apply. As Farmer John really hates racists, help Bessie identify which cows are racists based on their application. Input Format The first line of the integer will contain N. The second line will contain N integers, being the favorite number of applicant I. Output Format Print out N lines. For line I, print out YES if cow I is racist and NO if cow I is not racist. Sample Input 2 1 2 Sample Output YES NO Cow 1, who's favorite number was 1, was a racist. Cow 2, who's favorite number was 2, was not a racist. Scoring Inputs 1-5: The favorite number of no two applicants are the same. Inputs 6-10: No further restrictions. P1 P2 P3 P4 P5 P6 P7 P8 P9 P10
